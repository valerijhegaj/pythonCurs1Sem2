main - центральная часть всей програмы, в нее собираются все функции,
interface - реализация требуемого взаимодействия с программой в условии,
при запуске программы запускать interface
Специализация команд
1.exit - завершает работу программы
2.caesar [-e | -d] path [shift]
    -e - шифрование шифром
    -d - дешифрование
    path - путь до файла
    shift - сдвиг шифрования
3.HackCaesar path
    path - путь до дешифруемого файла (если файл маленький или
           шифрован не шифром цезаря, то может не сработать, 
           используется частотный анализ)
4.vigener [-e | -d] path keyWord
    -e - шифрование
    -d - дешифрование
    path - путь до файла
    keyWord - ключевое слово
5.vernam [-e | -d] path pathToKey
    -e - шифрование
    -d - дешифрование
    path - путь к файлу
    pathToKey - путь к ключу (должен совпадать по длине с файлом)
6.vernam [-ek | -dk] path key
    -ek - шифрование
    -dk - дешифрование
    path - путь к файлу
    key - ключ вбитый в консоль (должен совпадать по длине с файлом)
7.vernam -eg path pathToKey
    -eg - шифрование с генерацией ключа
    path - путь с файлом для шифрования
    pathToKey - путь к файлу в который будет записан ключ
                (в файле будет стерто содержимое и перезаписан ключ)
8.vernam -egk path
    -egk - шифрование с генерацией ключа с выводом ключа в консоль
    path - путь с файлом для шифрования
